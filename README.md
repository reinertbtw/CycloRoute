## Sobre o projeto

O **CycloRoute** é um aplicativo de navegação desenvolvido para ciclistas e usuários de scooters elétricas.

Diferente dos aplicativos tradicionais, o CycloRoute prioriza **ciclofaixas, ciclovias e rotas mais seguras**, oferecendo uma experiência de navegação pensada especialmente para mobilidade urbana leve.

## Pré-requisitos

Antes de começar, você precisará ter instalado:

- Flutter SDK
- Dart SDK
- Android Studio
- Emulador Android ou dispositivo físico
- Git

---

## Executando o projeto

Clone o repositório:

```bash
git clone https://github.com/reinertbtw/CycloRoute.git
```

Entre na pasta:

```bash
cd CycloRoute
```

Instale as dependências:

```bash
flutter pub get
```

Execute o aplicativo:

```bash
flutter run
```

## Objetivo

Criar uma alternativa ao Google Maps focada em:

- Ciclistas
- Usuários de scooters elétricas
- Mobilidade urbana sustentável
- Rotas mais seguras
- Melhor aproveitamento de ciclovias e ciclofaixas
